box

Set the script box style and attributes. Script box is where the dialogs are displayed.

Syntax
box, {param}
Configure the script box using the specified parameters. There is no need to call this if using the default settings.
Parameters are given as {param:value, param:value, ...}
Parameter Summary
Attributes Parameter Name and Description
Optional, Persistent
pos:"position_name"
Sets the position of the script box.
Valid values are "bottom" (default), "center", "top", "full.
BOTTOM - at bottom, size is 3/4 viewport width by 1/4 viewport height.
CENTER - at center, size is 3/4 viewport width by 1/4 viewport height.
TOP - at top, size is 3/4 viewport width by 1/4 viewport height.
FULL - at center, size is 3/4 viewport width by 7/8 viewport height.
BALLOON - (coming soon) Update: See 'actor'
Update: As of version 0.1.2, the box dimensions can be set in configuration file.
Optional, Persistent
back:"back_style"
Specify the background style of the box.
Valid values are "dim" (default), "none", "image".
DIM - script box shows a dim box with 50% opacity.
NONE - script box has no back style.
IMAGE - an image is displayed as a background.
Update: The DIM box colors and IMAGE file can be set in configuration file.
Optional, Persistent
align:"left", "center", "right"
Sets the text alignment for the box. Default is "left".
Optional, Persistent
prompt:"path_to_image_file" or "none"
Displays a small image at end of each text to indicate that a user input (i.e. click) is needed.
Optional
show:boolean (true/false)
Shows or hides the script box. Whenever there's a new text to be displayed, the script box is automatically shown.

Example

The script box is configured as given below. 
  
chapter = [

	...
	/* An empty dimmed script box is shown at center;
	note, no quotes on true */
	box, {pos:"center", back:"dim", show:true},
	...
	/* The script box is hidden */
	box, {show:false},
	...
	/* Adds a prompt and sets position to full, 
	still with dimmed back but box is not auto-shown 
	since there are no text to display anyway */
	box, {pos:"full", prompt:"images/prompt.png"}
	...
];
